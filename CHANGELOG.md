# Changelog
All notable changes to the Alogram PayRisk Ruby SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.9] - 2026-04-21
### Added
- **Initial Ruby Release**: Fully featured API client with automated model validation.
- **Enterprise Networking**: Implemented `httpx` core with connection pooling and HTTP/2 support.
- **Compliance Logic**: Integrated forensic traceability headers (`x-trace-id`) and Apache-2.0 licensing.
- **Async Readiness**: Native support for Ruby Fibers and non-blocking IO.
- **Identity Intelligence**: Forensic mapping for Telnyx v2 phone data.
